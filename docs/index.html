Pomodori

Stay focused, get more done

A free, open source Pomodoro timer that helps you break work into focused intervals. No accounts, no tracking, no distractions. Just you and deep work.

GitHub

Here's how it works

Simple by design. Powerful by habit.

Set your timer

Pick a focus interval that works for you. Classic 25 minutes or customize it to match your flow.

Do deep work

Block out distractions and focus on what matters. A gentle nudge keeps you on track until your break.

Take a break, repeat

Short breaks recharge your brain. After four rounds, take a longer break and come back refreshed.

Getting Started

Build from source and run it in under a minute.

Prerequisites

  • macOS 13 (Ventura) or later
  • Swift 5.9+ (included with Xcode 15+)

Build & Run from the Command Line

git clone https://github.com/mamartin/pomodori.git
cd pomodori
swift build
swift run

Build & Run with Xcode

open Package.swift
# Select the Pomodori scheme, choose "My Mac", then press Cmd+R

The app runs as a menu bar item — look for the tomato icon in your macOS menu bar.

Usage

Start / Pause / Resume Control a 25-minute work session
Reset Restart the current timer
Skip Jump to the next session
Quit Exit the app
